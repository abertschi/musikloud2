#MusiKloud2-InternetRadio

MusiKloud2-InternetRadio is a plugin for MusiKloud2 providing access to internet radio stations.

#Dependencies

python-simplejson

#Usage

The following resources are supported:

<table>
    <tr>
        <th>Resource</th>
        <th>Methods</th>
    </tr>
    <tr>
        <td>track</td>
        <td>list, search, get</td>
    </tr>
    <tr>
        <td>category</td>
        <td>list</td>
    </tr>
    <tr>
        <td>stream</td>
        <td>list</td>
    </tr>
</table>
