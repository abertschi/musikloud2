#MusiKloud2-Mixcloud

MusiKloud2-Mixcloud is a plugin for MusiKloud2 providing access to content from Mixcloud.

#Dependencies

python-simplejson

#Usage

The following resources are supported:

<table>
    <tr>
        <th>Resource</th>
        <th>Methods</th>
    </tr>
    <tr>
        <td>track</td>
        <td>list, search, get</td>
    </tr>
    <tr>
        <td>category</td>
        <td>list</td>
    </tr>
    <tr>
        <td>artist</td>
        <td>search, get</td>
    </tr>
    <tr>
        <td>stream</td>
        <td>list</td>
    </tr>
</table>
